# Integrated Management System

A desktop application developed using VB.NET Windows Forms for managing hospital billing and student loan processing.

## 📌 Project Overview
This system integrates two real-world modules into one application:
- Hospital Billing System
- Student Loan Management System

It includes user authentication, automated calculations, and a dashboard for navigation.

---

## 🏥 Hospital Billing Module
This module calculates patient bills based on:
- Ward type (General, Semi-Private, Private)
- Number of days admitted
- Additional medical services (e.g. lab tests, X-ray, surgery)
- VAT (12%)

It generates a final bill and flags premium billing cases.

---

## 🎓 Student Loan Module
This module:
- Checks loan eligibility based on GPA and family income
- Calculates monthly loan repayment using amortization
- Displays total repayment and monthly installment
